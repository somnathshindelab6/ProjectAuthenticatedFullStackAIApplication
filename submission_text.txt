Written Reflection
Accomplishment:
This project successfully combines a React frontend, a Flask backend, SQLAlchemy models, JWT authentication, and an AI-powered RAG workflow into a working full-stack application. The app allows users to register, log in, manage tasks, and receive AI-assisted recommendations in a secure environment.

Opportunity for Growth:
The biggest lesson learned was the importance of planning the architecture carefully and building the application in smaller, testable steps. Breaking the work into authentication, CRUD, and AI integration milestones made development and debugging easier and improved the overall structure of the project.

Continual Improvement:
If more time were available, the next improvements would focus on refining the UI, improving the quality of the AI knowledge base, strengthening security and authorization, and adding richer task personalization and user feedback features.
